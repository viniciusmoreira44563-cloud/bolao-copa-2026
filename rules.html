📋 Regras de pontuação

Sistema simples, direto e fácil de entender.

Tipo de acerto Pontuação Descrição Exemplo: Brasil 2 x 1 Sérvia
Placar exato 10 pts Acertou os gols dos dois times. Palpite: 2 x 1
Resultado simples 5 pts Errou o placar, mas acertou quem venceu ou que seria empate. Palpite: 1 x 0, 3 x 1, 3 x 2
Erro total 0 pts Errou quem venceria ou errou o empate. Palpite: 0 x 1 ou 1 x 1

🤝 Regra para empate

Empate exato: se o jogo foi 1 x 1 e o palpite foi 1 x 1, vale 10 pontos.

Empate simples: se o jogo foi 1 x 1 e o palpite foi 0 x 0 ou 2 x 2, vale 5 pontos.
