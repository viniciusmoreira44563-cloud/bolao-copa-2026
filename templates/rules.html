🏆 Como funciona a pontuação

🎯 Placar exato 10 pts

Acertou os gols dos dois times.


Exemplo: Brasil 2 x 1 Sérvia

Palpite: 2 x 1

✅ Resultado simples 5 pts

Errou o placar, mas acertou quem venceu ou que seria empate.


Exemplos: 1 x 0 • 3 x 1 • 3 x 2

❌ Erro total 0 pts

Errou quem venceria ou errou o empate.


Exemplos: 0 x 1 • 1 x 1

🤝 Regra para empate

🎯 Empate exato

Se o jogo foi 1 x 1 e o palpite foi 1 x 1:


Vale 10 pontos

✅ Empate simples

Se o jogo foi 1 x 1 e o palpite foi 0 x 0 ou 2 x 2:
